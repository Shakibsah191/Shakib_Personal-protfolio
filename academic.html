Academic Projects

Throughout my academic journey, I have worked on several projects that reflect my growing skills and interests in both software and hardware development. Here are three of my most notable projects:

Wise Spender – A personal finance management system built with HTML, CSS, JavaScript, PHP, and MySQL. It features income/expense tracking, pie chart visualizations, and login/logout functionality using a local database.

Netflix Clone – A front-end clone of the Netflix homepage using HTML and CSS. This project helped strengthen my layout design skills, responsive styling, and overall UI/UX understanding.

Smart Dustbin – A hardware-software integrated project focused on waste management. It detects waste levels and sends real-time data to concerned authorities, aiming to prevent overflow and support a cleaner environment.

These projects have not only boosted my technical abilities but also fueled my motivation to take on more complex and impactful challenges in the future.
